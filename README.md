To-Do List com React, Context API e React Router

Este projeto é uma aplicação completa de gerenciamento de tarefas (To-Do List), construída em React utilizando:

✔ React Router — múltiplas páginas
✔ Context API — estado global
✔ useState/useEffect — hooks fundamentais
✔ localStorage — persistência dos dados
✔ CRUD completo — criar, listar, editar e excluir tarefas

 Funcionalidades

Adicionar novas tarefas

Listar todas as tarefas

Editar tarefa existente

Excluir tarefas

Persistência automática no localStorage

Navegação entre páginas usando React Router

 Tecnologias utilizadas

React

React Router DOM

Context API

JavaScript ES6

localStorage

Como Rodar o Projeto
1. Instale as dependências
npm install

2. Execute o projeto
npm start


A aplicação abrirá em:

http://localhost:3000
